# Restart V12.3.4

本版將角色預覽改為固定 1024 × 1024 邏輯畫布的 Canvas Avatar Engine。

## 本版完成

- 共用身體、頭型、男女初始表情
- 男生初始髮型 001（前髮／後髮）
- 女生初始髮型 001（前髮／後髮）
- 固定角色中心 X=512
- 固定腳底約 Y=920
- 頭部與身體分組校正，不再直接用 CSS 任意縮放
- 選角、首頁、衣櫃、商城縮圖使用同一個渲染器
- 點角色互動為揮手

## 素材規格

所有素材均使用 1024 × 1024 透明 PNG，保持相同畫布與座標。


## V12.3.5

- 商城新增 8 種基本髮色，全部預設 1 戒賭幣。
- 髮色使用 RGB 作為資料 ID：黑、紅、橘、黃、綠、藍、紫、棕。
- 棕色 RGB(155,103,55) 為初始免費髮色。
- 其餘 7 色購買後加入衣櫃，並從商城消失。
- 同一張棕色髮型 PNG 由 Canvas 即時染色，保留原本透明度、陰影與高光。
- 商城縮圖、試穿角色、衣櫃與首頁使用同一個染色渲染結果。
