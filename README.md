# Restart V13.5

本版更新：
- 管理員入口只保留在初始登入頁底部，與一般會員登入分離。
- 商城改用 Supabase RPC 真正扣除戒賭幣，並留下交易紀錄。
- 新增 5 項每日任務與雲端領獎紀錄。
- 主城廣場改用新的 MMORPG 背景。
- 廣場支援左右拖曳、人物移動時鏡頭跟隨。
- 圖中建築已設定透明入口：公告欄、公會、商城、傳送門、每日任務、鍛造屋、涼亭。

## Supabase
請在 SQL Editor 執行 `supabase-v13.5.sql`。可以貼到原本的合併 Query 最下方再執行，也可以使用 New query；兩種方式的資料庫效果相同。
