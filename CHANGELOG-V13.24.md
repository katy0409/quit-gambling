# Restart V13.24 修改紀錄

## 角色初始服裝
- 新增男女共用的初始上衣與初始下裝底圖（正面／左／右）。
- 初始上衣、初始下裝不列入衣櫃，也不是商城商品。
- 穿上其他上衣時隱藏初始上衣；卸下後恢復初始上衣。
- 穿上其他下裝時隱藏初始下裝；卸下後恢復初始下裝。
- 穿著套裝時隱藏初始上衣、初始下裝及一般上下裝。
- 原本的白色 T 恤與黑色短褲改為商城購買商品，購買後才加入衣櫃。

## 角色建立畫面
- 角色卡片加高，完整顯示鞋子，不改人物寬度與男女並排版面。
- 膚色選項由 5 種增加為 8 種：瓷白、白皙、自然白、自然、小麥、健康、古銅、深膚。
- 男女角色預覽同步套用所選膚色。

## 記帳頁
- 優化今日／本月六格統計卡片。
- 固定三欄等寬，長金額維持單行並自動縮放。
- 使用等寬數字，避免金額變動造成版面跳動。

## 版本與資料庫
- 版本更新為 V13.24。
- 本次未修改資料庫，未附 SQL。


## V13.24 行動裝置角色預覽修正
- 修正角色選擇卡片中腳部與鞋子被裁切的問題。
- 擴大初始角色預覽的畫布取景範圍，完整顯示至鞋底。
- 行動裝置角色卡片與畫布高度同步增加。
- 新增 380px 以下窄螢幕調整，維持男女角色左右並排且不切腳。
- 更新 CSS、Avatar JavaScript 與 Service Worker 快取版本。

## V13.24 初始鞋子調整
- 初始男女角色改為赤腳，不再自動穿著鞋子。
- 原本的白色休閒鞋改為男女共用商城商品，購買後才加入衣櫃。
- 白色休閒鞋不再列入新手免費持有物品。
- 既有玩家若仍由舊版免費持有或穿著白色休閒鞋，更新後會移除並恢復赤腳；之後可由商城重新購買。
- 脫下鞋子後維持赤腳，不會自動補回預設鞋。
- 本次未修改資料庫，未附 SQL。
