# Restart V13.23 修改紀錄

## SOS
- 「呼吸 1 分鐘」改為「呼吸 5 分鐘」，啟動後倒數 5 分鐘。
- 移除「重新計時 5 分鐘」按鈕，避免與呼吸計時功能重複。
- 「LINE 戒賭社群」文字改為「前往接渡有緣人」，原連結邏輯維持不變。
- 新增獨立 `js/sos-quotes.js`，內含 300 句戒賭金句。
- 每次開啟 SOS 隨機顯示一則，並以本機紀錄避免連續顯示同一句。

## 角色系統
- 角色 Canvas 圖層順序統一為：Base → Expression → Shoes → Bottom → Top → Accessory（預留）→ Hair。
- 初始角色精簡為男生一位、女生一位。
- 上衣、下裝與鞋子維持男女共用；僅髮型依角色性別區分。
- 移除初始角色清單中未完成的多餘角色，避免舊卡片與覆寫邏輯衝突。

## 廣場
- 可走範圍由 Y 72–91 調整為 Y 76–94，整體往下移。
- 初始位置同步下移至 Y 87。
- 舊有儲存位置載入時會重新限制在新版可走範圍內。
- 玩家名稱固定顯示在人物下方並跟隨人物移動。
- 名稱改為白字、黑色描邊陰影、無背景框。
- 新增最終覆寫規則，統一舊版多段 `.square-name` CSS，避免互相覆蓋。

## 版本與快取
- 靜態資源版本參數與 Service Worker 快取更新為 V13.23。
- Service Worker 預快取加入 `js/sos-quotes.js`。

## 資料庫
- 本次未修改資料庫結構，因此未附 SQL 檔。

## 保持原樣
- 未重新設計任何版面或未指定功能，其餘 V13.22 功能與資料邏輯保持原樣。

## 追加修正
- 初始角色選擇改為男生、女生左右並列。
- 顯示 5 種膚色選項，選擇後角色預覽立即同步更新。
- 膚色會在確認角色後保存。
- 修正廣場角色舊位移 CSS 導致玩家名字未顯示的問題。
